Testing HCFS replication

If you would like to test HCFS replication using larger and more complicated data sets, you can run through these examples.

TeraGen

Run the following command to generate 10GB of data inside your replicated path:

hadoop jar hadoop-*examples*.jar teragen 100000000 /path/to/replication-rule
